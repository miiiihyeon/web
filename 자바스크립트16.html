빌트인 객체 Native object, Host object, User-defined object로 분류 1. Native object - ECMAScript 명세에 정의된 객체. 애플리케이션 전역의 공통 기능 제공, 애플리케이션 환경과 상관없이 사용 가능 - Global object. !==전역객체 1.1 Object -Objec() 생성자 함수는 객체를 생성. 만약에 생성자 인수값이 null이거나 undefined이면 빈 객체를 반환 - 생성자 함수의 인수값에 따라 강제 형변환된 객체가 반환 1.2 Function - Function객체는 new연산자를 사용해 생성 1.3 Boolean - Boolean 생성자 함수로 Boolean 객체 생성 var foo = new Boolean('false'); //true Boolean 객체는 true/false를 포함하고 있는 객체 1.4 error - Error 생성자는 error 객체를 생성. error 객체의 인스턴스는 런타임 에러가 발생했을 때 throw 1.5 symbol - 유일하고 변경 불가능한 원시타입. symbol객체는 원시타입 symbol 값을 생성 1.6 원시타입과 래퍼객체 - Native 객체는 각자의 프로퍼티와 메소드를 가짐. - 정적 프로퍼티, 메소드는 해당 인스턴스를 생성하지 않아도 사용할 수 있고, prototype에 속해있는 메소드는 해당 prototype을 상속받은 인스턴스가 있어야 사용 but, 원시 타입 값에 대해 표준 빌트인 객체의 메소드를 호출하면 정상적 작동 ( 원시 타입 값에 대해 표준 빌트인 객체의 메소드를 호출할 때, 원시 타입 값은 연관된 객체로 일시 변환되기 때문에 ) -> 메소드 호출이 종료되면 객체로 변환된 원시 타입은 다시 원시타입 값으로 복귀. - String, Number, Boolean 2. Host object - DOM노드 객체와 같이 호스트 환경에 정의된 객체. 2.1 전역객체 - 모든 객체의 유일한 최상위 객체 Browser-side : window, Server-side : global 2.2 BOM (Browse object model) - 브라우저 탭 또는 브라우저 창의 모델 생성 최상위 객체 :window * document : 현재 로드된 웹페이지 * history : 브라우저 히스트리에 기록된 웹페이지 * location : 현재 페이지 url * navigator : 브라우저 관련 정보 * screen : 장치의 디스플레이 정보 2.3 DOM (Document object model) -현재 웹페이지의 모델을 생성 . 최상위 객체 : document
